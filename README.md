### Instalação

Instale as dependecias para rodar o servidor
```sh
$ cd Quickjob
$ npm install
```

Para iniciar em modo de desenvolvimento utilize o comando
```sh
$ gulp develop --store=medgrupo --path=store/medgrupo/appweb
```
